ArcGIS License Manager は、ArcGIS Desktop、ArcGIS Engine、ArcGIS CityEngine、ArcGIS Pro の同時使用ライセンス、および ArcGIS Enterprise を経由する ArcGIS Pro またはプレミアム アプリの指定ユーザー ライセンスのインストールと管理をするための製品です。

{{< callout type="important">}}

米国 Esri は、同時使用ライセンス タイプおよび ArcGIS License Manager のサポート終了を発表しました。<br>
2026 年 2 月末以前にリリースされたバージョンでは、既存の同時使用ライセンスは引き続き使用可能ですが、2026 年 3 月 1 日以降にリリースされる ArcGIS Pro および ArcGIS CityEngine では、同時使用ライセンスはサポートされません。
詳細は「[サポート終了情報: 同時使用ライセンス タイプと ArcGIS License Manager について](https://tech-support.esrij.com/arcgis/article/web/knowledge5996.html)」をご参照ください。

{{< /callout >}}
