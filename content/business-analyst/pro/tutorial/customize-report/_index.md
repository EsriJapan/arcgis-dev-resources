# 目的

ユーザー独自のインフォグラフィックス テンプレートおよびクラシック レポート テンプレートを作成し、ローカル データセットを使用したレポートの作成方法を習得します。

{{< callout >}}

レポートのカスタマイズには **ArcGIS Pro 3.2** 以降のバージョンが必要です。
