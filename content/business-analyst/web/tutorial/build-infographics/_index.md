BA Web App では、<a href="https://business-map.esrij.com/glossary/4328/" target="_blank">インフォグラフィックス</a>などの商圏レポートを実行し、商圏内の人口や世帯の分布を簡単操作で調べることができます。また、BA Web App Advanced ライセンスをお持ちであれば、独自のレポート テンプレートを構築し、組織内に共有して商圏分析に利用いただけます。

本演習では、インフォグラフィックスのカスタマイズ方法を習得します。

{{< callout type = "warning" >}}

商圏レポートのカスタマイズやカスタム データの設定を行うためには、ArcGIS Business Analyst Web App Advanced ライセンスが必要です。
