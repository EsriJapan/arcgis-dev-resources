ArcGIS Online 上にあるフィーチャ レイヤーやシーン レイヤーなどを Scene Viewer に追加して、3D シーンの表示と操作が可能です。シーンの保存には、コンテンツを作成する[権限](/online/get-started/administer/members/)が必要です。

### Scene Viewer の起動
1. ArcGIS Online 組織サイトの右上に表示されている [アプリ ランチャー] → [Scene Viewer] をクリックします。
{{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/online/users-guide/scene/create-start.png" title=" " width="300" >}}

2. [新しいシーン] をクリックします。
{{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/online/users-guide/scene/create-newscene.png" title=" " width="400" >}}

3. デフォルトでは、球体の 3D ビューアーが起動します。

### グローバル/ローカル シーンの選択
1. 右上の [新しいシーン] をクリックして、[新しいグローバル シーン] または [新しいローカル シーン] を選択します。
{{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/online/users-guide/scene/create-scene-choice.png" title=" " width="200" >}}

2. [新しいグローバルシーン] では球体の、[新しいローカル シーン] では平面の 3D ビューアーが起動します。
{{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/online/users-guide/scene/create-global-local.png" title=" " width="600" >}}

{{< callout >}}

ローカル シーンでは [プロパティ] → [指定範囲にクリップ] でシーンの表示範囲をクリップすることができます。詳細は[ヘルプ](https://doc.arcgis.com/ja/arcgis-online/create-maps/set-scene-properties.htm#ESRI_SECTION1_8FAC7522456A4258B9243257D4A201FC)をご参照ください。

{{< /callout >}}

### レイヤーの参照
1. [コンテンツ] ツールバーの [レイヤーの追加] をクリックします。
1. [レイヤーの参照] をクリックします。
{{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/online/users-guide/scene/create-add-layer.png" title=" " width="700" >}}
1. [マイ コンテンツ] 横の矢印をクリックし、検索する場所を指定します。
{{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/online/users-guide/scene/create-browse-layer.png" title=" " width="300" >}}
1. [レイヤーの検索] ボックスにキーワードを入力し、Enter キーで検索します。
1. [フィルター] をクリックすると、[フィルター] パネルが開きます。[タイトル] や [関連性] などで検索結果の並べ替えや、[更新日] や [アイテム タイプ] などでフィルターして検索することも可能です。
1. 検索結果の [追加] をクリックして、レイヤーをマップに追加します。追加後に [削除] をクリックしてマップから削除することもできます。
{{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/online/users-guide/scene/create-search-layer.png" title=" " width="400" >}}
1. [閉じる] をクリックして、パネルを閉じます。


{{< callout >}}

[ArcGIS Web サービス]、[GeoJSON]、[CSV]、[見通し線]、[可視領域]、[ディメンション] レイヤーの追加手順については[「レイヤーをシーンに追加」](https://doc.arcgis.com/ja/arcgis-online/create-maps/add-layers-to-scene.htm) をご参照ください。

{{< /callout >}}


### シーンの保存
1. [保存] をクリックします。
2. [タイトル]、[サマリー]、[タグ] に適切な文字を入力し、[保存] をクリックします。
{{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/online/users-guide/scene/create-save.png" title=" " width="500" >}}
