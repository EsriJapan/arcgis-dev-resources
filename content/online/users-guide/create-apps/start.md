Web アプリの作成を開始する手順は、以下の A、B、C の 3 つの方法があります。どの方法から開始しても、アプリの作成が可能です。作業しやすい開始手順でアプリの作成を開始してください。


### A. Map Viewer から作成
1. Map Viewer で作成した Web マップを保存します。
2. [アプリの作成] をクリックします。
3. 作成したい Web アプリを選択します。
{{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/online/users-guide/create-apps/start-method-a1.png" title="方法 A1" width="400" >}} 
{{< callout >}}

左下の [展開] をクリックすることで、各タブの名称を表示することができます。

{{< /callout >}}

{{< callout >}} 

ご使用のディスプレイのサイズが小さい場合、[その他] に [アプリの作成] が格納されています。
{{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/online/users-guide/create-apps/start-method-a2.png" title="方法 A2" width="250" >}} 

{{< /callout >}}


### B. アイテム詳細ページから作成
1. [コンテンツ] タブをクリックし、 [マイ コンテンツ] を表示します。
{{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/online/users-guide/create-apps/start-method-b1.png" title="方法 B1" width="750" >}} 

2. Web アプリに使用したい Web マップのタイトルをクリックして、アイテム詳細ページを開きます。
{{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/online/users-guide/create-apps/start-method-b2.png" title="方法 B2" width="500" >}}

3. 画面右側の [Web アプリの作成] をクリックし、作成したい Web アプリを選択します。
{{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/online/users-guide/create-apps/start-method-b3.png" title="方法 B3" width="300" >}}

### C. マイ コンテンツから作成
1. [コンテンツ] タブをクリックし、[マイ コンテンツ] を表示します。
2. 画面左上にある [アプリの作成] をクリックし、作成したい Web アプリを選択します。
{{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/online/users-guide/create-apps/start-method-c1.png" title="方法 C1" width="500" >}}
