アプリとは、ArcGIS Online で作成するマップ (Web マップ) に目的に応じて機能や見栄えに変更を加えたアプリケーションです。Web アプリは、主に以下の 4 種類があります。  
なお、Web アプリの作成には、コンテンツの「作成、更新、および削除」[権限](https://doc.arcgis.com/ja/arcgis-online/reference/roles.htm)が必要です。

[1. ArcGIS Instant Apps](#arcgis-instant-apps)  
[2. ArcGIS Experience Builder](#arcgis-experience-builder)  
[3. ArcGIS StoryMaps](#arcgis-storymaps)  
[4. ArcGIS Dashboards](#arcgis-dashboards)   


### ArcGIS Instant Apps
マップとデータを簡単に操作できるようにする Web アプリをテンプレートから選び、Web アプリをすばやく作成します。<br />
詳細は[製品ページ](https://www.esrij.com/products/arcgis-instant-apps/)をご参照ください。
{{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/online/users-guide/create-apps/webapp-instantapps.png" title="ArcGIS Instant Apps" width="750" >}} 

### ArcGIS Experience Builder
2D/3D マップ、テキスト、メディアを含む、レスポンシブ対応の Web アプリやページを作成します。<br />
詳細は[製品ページ](https://www.esrij.com/products/arcgis-experience-builder/)をご参照ください。
{{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/online/users-guide/create-apps/webapp-exb.png" title="ArcGIS Experience Builder" width="500" >}}  

### ArcGIS StoryMaps
マップにテキスト、画像、動画などを組み合わせた Web ページを作成します。<br />
詳細は[製品ページ](https://www.esrij.com/products/arcgis-storymaps/)をご参照ください。
{{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/online/users-guide/create-apps/webapp-storymaps.png" title="ArcGIS StoryMaps" width="750" >}} 

### ArcGIS Dashboards
マップ、グラフ、ゲージなどを組み合わせて情報を表示するダッシュボードを作成します。<br />
詳細は[製品ページ](https://www.esrij.com/products/arcgis-dashboards/)をご参照ください。
{{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/online/users-guide/create-apps/webapp-dashboards.png" title="ArcGIS Dashboards" width="750" >}}
