サイトを作成した後、サイトを編集モードで開き、画面左側に表示されているサイド パネルを使用して、サイトの構成を行います。 

### サイトを編集モードで開く

1. ArcGIS Hub のマイ ワークスペースのホーム画面で、作成したサイトのタイトルをクリックします。
   {{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/hub/get-started/site/configure-myworkspace-site.png" caption="サイトのタイトルをクリック" width="600">}}

1. [編集] をクリックします。
   {{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/hub/get-started/site/configure-open-editmode.png" caption="編集モードで開く" width="450" >}}

## サイド パネル内の各項目について

   {{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/hub/get-started/site/configure-side-panels.png" caption="サイド パネル" width="600" >}}<br>

- [設定](https://doc.arcgis.com/en/hub/sites/configure-site-settings.htm): サイトの名前やドメインなどを設定できます。	
- [テーマ](https://doc.arcgis.com/ja/hub/sites/brand-a-site-with-header-and-theme-options.htm#ESRI_SECTION1_D5F12DB0B6E341698B2715386CC7A5C7): サイト全体のテキスト色やボタンの色などを設定できます。
- [ヘッダー](https://doc.arcgis.com/ja/hub/sites/brand-a-site-with-header-and-theme-options.htm#ESRI_SECTION1_6C418A4D0C8840569C79C572C0A5632E): サイトのヘッダーへのロゴの追加や、スタイルなどの変更が可能です。
- [レイアウト](https://doc.arcgis.com/en/hub/sites/add-text-and-images.htm): サイトにテキスト、画像、アプリなどを表示するためのカードが提供されています。レイアウト上にカードをドラッグ&ドロップで追加・設定することにより、サイトにテキスト等を表示できます。
- [フッター](https://doc.arcgis.com/ja/hub/sites/brand-a-site-with-header-and-theme-options.htm#ESRI_SECTION1_A8A8E5B30120492EA85E531A7A46CEC8): カスタム HTML と カスタム CSS を使用して、サイトにフッターを追加できます。

## サイトのレイアウトの設計
1. サイド パネルの [レイアウト] メニューを開き、テキストや画像などのコンポーネントをサイトに追加できるカードを選択します。
1. サイト上にカードをドラッグ ＆ ドロップで配置します。
1. 配置したカードの移動・編集・削除はカード上にカーソルを移動すると表示される [移動]・[編集]・[削除] ボタンをクリックして行います。カードによっては、[編集] ボタンをクリックするとサイド パネルが表示されます。  
   {{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/hub/get-started/site/configure-move-edit-delete-cards.png" caption="サイトのレイアウト" width="800" >}}

   {{< callout type="info" >}}      

   カードの詳細については、[カードの配置と設定](/hub/get-started/site/design/card)をご参照ください。
