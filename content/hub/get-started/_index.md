**ArcGIS Hub** の利用を開始する方のためのガイドです。<br>
Web サイトやオープンデータ ポータルの構築や、コンテンツの管理の設定などについて説明します。

{{< callout type="warning" >}}

ArcGIS Hub を利用して Web サイトを作成するためには、ArcGIS Online 上でメンバーに[適切な権限](https://doc.arcgis.com/ja/hub/team/configure-roles-and-privileges.htm#ESRI_SECTION1_38901D2E06E9488BB706AF79E23A6A1B)を割り当てる必要があります。

{{< /callout >}}

{{< callout type="info" >}}

ArcGIS Online をはじめてご利用になる場合は、[ArcGIS Online スタートアップ ガイド](online/get-started)をご参照ください。
