ArcGIS Hub サイトでアイテムを共有するには、アイテムをコンテンツ カタログのグループに追加して共有範囲を設定する必要があります。

## コンテンツをグループに追加
ArcGIS Hub サイトで共有するアイテムはグループ単位で管理できます。共有したいアイテムをあらかじめ作成したグループに追加します。

1. ArcGIS Hub のマイ ワークスペースのホーム画面で、[グループ] をクリックし、[マイ グループ] ページを開きます。
1. 「<サイトのタイトル> Content」グループの [管理] をクリックして、グループの [詳細] ページを開きます。

    {{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/hub/get-started/content/add-content-catalog-my-group-page.png" caption="マイ グループ ページ" width="650" >}}

    {{< callout type="info" >}}

    [サイトの作成](https://doc.esrij.com/hub/get-started/site/create-site#サイトの作成)時に [空白のカタログ] または [既存のグループによるカタログ] オプションを選択して作成した場合は、任意のグループを選択します。
    
    {{< /callout >}}

1. [コンテンツ] → [コンテンツの追加] をクリックし、[コンテンツの追加] パネルを開きます。

    {{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/hub/get-started/content/add-content-catalog-open-content-panel.png" caption="コンテンツの追加パネルを開く" width="500" >}}

1. 追加したいアイテムを選択し、[追加] をクリックしてアイテムをグループに追加します。

    {{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/hub/get-started/content/add-content-catalog-add-content-group.png" caption="コンテンツをグループに追加" width="550" >}}

## コンテンツ カタログの構成 (オプション)
コンテンツ カタログを構成していない ArcGIS Hub サイトで、カタログを新しく構成するための操作を説明します。</br>

{{< callout type="info" >}}

サイトの作成時に [カタログをクイック スタート] オプションを選択して作成した場合、このセクションは必要ありません。

{{< /callout >}}

1. ArcGIS Hub のマイ ワークスペースのホーム画面で、[コンテンツ] をクリックし、[コンテンツ] ページを開きます。
1. サイトの [管理] をクリックし、サイトの [詳細] ページを開きます。

 
    {{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/hub/get-started/content/add-content-catalog-open-detail-page.png" caption="コンテンツ ページ" width="650" >}}

1. [カタログ] ドロップダウンの [コンテンツ] をクリックし、[カタログの構築] をクリックします。

    {{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/hub/get-started/content/add-content-catalog-click-configure-catalog.png" caption="カタログの構築をクリック" width="650" >}}

1. 画面右側の [カタログの構成] パネルで [フィルターの設定] を展開し、[グループの選択] をクリックします。
1. [グループの選択] パネルで、[コンテンツをグループに追加](https://doc.esrij.com/hub/get-started/content/add-content-catalog/#コンテンツをグループに追加)セクションでコンテンツを追加したグループを選択し、[追加] をクリックします。

    {{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/hub/get-started/content/add-content-catalog-add-group-catalog.png" caption="カタログにグループを追加" width="700" >}}

2. [コンテンツ] ページでプレビューを確認し、[変更の保存] をクリックしてカタログの構成を保存します。

    {{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/hub/get-started/content/add-content-catalog-save-catalog-setting.png" caption="カタログの構成を保存" width="300" >}}
