ArcGIS CityEngine の[インストール](#インストール)と[アンインストール](#アンインストール)の手順について説明します。

## インストール
ArcGIS CityEngine をインストールする手順を説明します。

1. CityEngine インストール プログラムである Setup.exe を起動します。
1. Welcome 画面のインストール作業上の注意事項を読みます。内容を確認したら [Next] をクリックします。
1. マスター契約書 (Master Agreement) を読み、内容に同意したら [I accept …] を選択して [Next] をクリックします。
1. [Change] をクリックしてインストール先のフォルダーを指定するか、デフォルトの場所で良ければそのまま [Next] をクリックします。
1. [Install] をクリックしてインストールを開始します。
1. [Finish] をクリックしてウィザードを終了します。

{{< callout >}}

インストール完了後、CityEngine Software Authorization Wizard (ソフトウェア認証ウィザード) が起動します。詳細は「[ソフトウェアの認証]({{% ref "/cityengine/get-started/setup/install/authorization" %}} "ソフトウェアの認証")」をご参照ください。

{{< /callout >}}

## アンインストール
ArcGIS CityEngine をアンインストールする手順を説明します。

{{< callout >}} 

単独使用 (SU) ライセンスの場合は、必ずアンインストール前にライセンスの認証解除を行ってください。

{{< /callout >}}

- [コントロール パネル] を使用する方法
1. Windows の [コントロール パネル] → [プログラムと機能] をクリックします。
1. ArcGIS CityEngine をクリックして [アンインストールと変更] から [削除] オプションを選択します。

- [設定] を使用する方法
1. [スタート] → [設定] → [アプリ] から [アプリと機能] へ移動します。
1. ArcGIS CityEngine をクリックし、[アンインストール] から [削除] オプションを選択します。
