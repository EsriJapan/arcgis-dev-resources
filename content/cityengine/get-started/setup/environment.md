## システム要件の確認

ArcGIS CityEngine をインストールするコンピューターがシステム要件を満たしているかどうかをご確認ください。

システム要件の詳細は、「[CityEngine system requirements](https://doc.arcgis.com/en/cityengine/latest/get-started/cityengine-system-requirements.htm) (英語)」ページをご参照ください。

{{< callout >}}

同時使用 (Concurrent Use) ライセンスをご利用の場合は、[ArcGIS License Manager のシステム要件](https://desktop.arcgis.com/ja/license-manager/latest/arcgis-license-manager-system-requirements.htm)もご参照ください。

{{< /callout >}}
