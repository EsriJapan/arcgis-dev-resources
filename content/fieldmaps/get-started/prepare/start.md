ArcGIS Field Mapsには、Field Maps モバイル アプリと Field Maps Web アプリの 2 つのアプリが付属しています。<br>
Field Maps Web アプリでは、モバイル作業者が Field Maps モバイル アプリで使用する Web マップおよびフィーチャ レイヤーを構成できます。

### Web アプリの起動

1. ArcGIS Online 組織サイトの右上に表示されている [アプリ ランチャー] → [Field Maps Designer] をクリックします。

{{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/fieldmaps/get-started/prepare/start-app.jpg" title="Web アプリの起動" width="450" >}}

2. 現地調査で利用するフィーチャ レイヤーおよび 、Web マップを Web アプリ上で作成する場合は [新規マップ] をクリックし、「[Web マップの作成](/fieldmaps/get-started/prepare/create-map)」に進みます。

{{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/fieldmaps/get-started/prepare/start-newmap.png" title="新規マップ" width="450" >}}
{{< callout >}}

Field Maps モバイル アプリでは既存の Web マップを利用することも可能です。目的の Web マップが表示されない場合は、ArcGIS Online で Web マップのアイテム ページを開き、[設定] → [アプリケーション設定] から [ArcGIS Field Maps で使用] チェックボックスをオンにしてください。

{{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/fieldmaps/get-started/prepare/start-setting.png" title="Web マップの設定" width="450" >}}
