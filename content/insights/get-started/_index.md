ロケーション アナリティクスと BI ワークフローを兼ね備えた分析用ワークベンチ **ArcGIS Insights** の利用を開始する方のためのガイドです。

## ArcGIS Insights の配置オプション
 
ArcGIS Insights では、以下の異なる環境に対応したアプリケーションを提供しています。

|環境|アプリケーション|
| ---- | ---- |
| クラウド| Insights in ArcGIS Online (ArcGIS Online 版) |
| オンプレミス| Insights in ArcGIS Enterprise (ArcGIS Enterprise 版) |
| デスクトップ| Insights desktop (デスクトップ版)|

{{< callout >}}

Insights desktop は、ArcGIS Insights (Insights in ArcGIS Online もしくは Insights in ArcGIS Enterprise) のライセンスをお持ちであれば、無償で利用いただけるアプリケーションです。

{{< /callout >}}

<a href="https://doc.arcgis.com/ja/insights/latest/get-started/get-started.htm" target="_blank" rel="noopener noreferrer">ArcGIS Insights ヘルプ ドキュメント</a>では、各環境が対応するページに下記のアイコンが表示されます。

## Insights の利用開始方法

### 1. ライセンスの割り当て (共通)

組織の管理者は、ArcGIS 組織のユーザーに対して ArcGIS Insights のライセンスの割り当てを行う必要があります。

<a href="https://doc.arcgis.com/ja/insights/latest/administer/administer-insights.htm" target="_blank" rel="noopener noreferrer">Insights in ArcGIS Enterprise および Insights in ArcGIS Online の管理</a>に、ユーザー タイプ、ロール、ライセンスを適用する方法へのリンクがございます。

### 2. セットアップ

#### Insights in ArcGIS Online

<a href="https://doc.arcgis.com/ja/insights/latest/administer/configure-online.htm" target="_blank" rel="noopener noreferrer">Insights をサポートするための ArcGIS Online の構成</a>を参照し、設定を確認します。

#### Insights in ArcGIS Enterprise

 <a href="https://www.esri.com/content/dam/esrisites/ja-jp/media/pdf/implementation-guides/implement-insights-enterprise.pdf" target="_blank" rel="noopener noreferrer">Insights in ArcGIS Enterprise 実装ガイド</a>に沿って、<a href="https://my.esri.com/" target="_blank" rel="noopener noreferrer">My Esri</a> からセットアップ ファイルのダウンロードを行い、インストールを行ってください。

{{< callout >}}

My Esri の利用に関する詳細は、<a href="https://www.esrij.com/myesri/" target="_blank" rel="noopener noreferrer">My Esri 利用手順 | ESRIジャパン</a> をご参照ください。

{{< /callout >}}

#### Insights desktop (任意)

<a href="https://doc.arcgis.com/ja/insights/latest/get-started/install-insights-desktop.htm" target="_blank" rel="noopener noreferrer">Insights desktop のインストール</a>の手順に沿って、セットアップ ファイルのダウンロードとインストールを行ってください。インストールが完了したら、Insights desktop を起動し、<a href="https://doc.arcgis.com/ja/insights/latest/get-started/activate-desktop.htm" target="_blank" rel="noopener noreferrer">Insights desktop のアクティブ化</a>を行います。

### 3. サイン イン (共通)

適切なライセンスの割り当て後は、Insights にサイン インすることが可能になります。

各アプリケーション でのサイン イン方法は、<a href="https://doc.arcgis.com/ja/insights/latest/get-started/access-insights.htm" target="_blank" rel="noopener noreferrer">Insights へのサイン イン</a>をご覧ください。


### 4. ワークブックでの操作 (共通)

サイン イン後、 [初めてのワークブック作成] をクリックし、分析を行うためのページ (= ワークブック) を起動します。

起動後の操作の概要は、<a href="https://doc.arcgis.com/ja/insights/latest/get-started/get-started-with-workbooks.htm" target="_blank" rel="noopener noreferrer">ワークブックの基本操作</a>をご覧ください。
