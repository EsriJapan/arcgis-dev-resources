ArcGIS Pro のライセンスとエクステンションは ArcGIS Online で管理されるため、ArcGIS Online の利用登録が必要です。<br>ArcGIS Online の登録とライセンスの割り当ては、複数のステップで構成されています。


<img src="https://s3-ap-northeast-1.amazonaws.com/doc.esrij.com/pro/get-started/setup/admin-setup.png" width="800" alt="ステップ">

ArcGIS Online の組織サイトを既にご利用の方は「[メンバーの追加]({{% ref "online/get-started/setup/admin/add-members" %}} "メンバーの追加")」からライセンスの割り当てを行ってください。<br>各ステップの詳しい手順については、各ステップのタイトルからリンクしている ArcGIS Online スタートアップ ガイドをご参照ください。

## 1. [サブスクリプションの有効化]({{% ref "online/get-started/setup/admin/activate-subscription" %}} "サブスクリプションの有効化")

「サブスクリプションの有効化」とは、ArcGIS Online のクラウドに参加するための手続きに相当します。<br> 
ご契約時に受信されたメールまたはライセンス コンファメーションに記載されている URL から <サブスクリプション ID 有効化ページ> にアクセスし、アカウントの設定を行います。 

{{< callout type="info" >}} 

保守契約が有効な ArcGIS Desktop ライセンスをお持ちで、組織内で ArcGIS Online が有効化されていない場合、ご契約時に受信されたメールまたはライセンス コンファメーションや保守登録完了通知内に記載の<お客様のサブスクリプション ID 有効化ぺージ>から ArcGIS Online を有効化してください。不明な場合は、[My Esri](https://tech-support.esrij.com/arcgis/article/web/knowledge101.html) からお問い合わせください。

{{< /callout >}}

## 2. [組織の設定]({{% ref "online/get-started/setup/admin/settings" %}} "組織の設定")

ArcGIS Online では組織専用のサイトを作成することができます。<br>
組織サイトの設定ページで、ショートネーム (https://○○.maps.arcgis.com の○○の URL 名を決定)、言語、地域、セキュリティを設定します。

## 3. [メンバーの追加]({{% ref "online/get-started/setup/admin/add-members" %}} "メンバーの追加")

組織サイトの構成を設定後、管理者は ArcGIS Pro を使用したい利用者を組織サイトに招待できます。<br>
組織サイトから、メンバーの [名]、[姓]、[電子メール アドレス]、サイン イン時に使用する [ユーザー名]、[ユーザー タイプ]、[ロール]、[一時パスワード] を設定します。

## 4. [ライセンスの割り当てと解除]({{% ref "online/get-started/administer/license/assign" %}} "ライセンスの割り当てと解除")

ArcGIS Pro で使用するライセンス (エディションおよびエクステンション) は、ArcGIS Online 組織ページより割り当てを行う必要があります。<br>
管理者の方は組織サイトから各メンバーにライセンスの割り当てを行ってください。

{{< callout type="info" >}}

ステップ 3 において、組織サイトへのメンバーの追加が確実に行われている必要があります。

{{< /callout >}}

{{< callout type="info" >}} 

ステップ 3 において、Creator、Professional、Professional Plus ユーザー タイプを設定している場合はライセンスの割り当ての必要はありません。

{{< /callout >}}

## 5. [組織サイトへ加入]({{% ref "online/get-started/setup/member/join" %}} "組織サイトへ加入")

ArcGIS Pro を利用するためには、ArcGIS Online の組織サイトのメンバーである必要があります。<br>
ArcGIS Online の組織サイトの管理者から招待を受けたメンバーは、メールに記載されている URL から組織サイトへアクセスし、組織サイトに加入してください。
