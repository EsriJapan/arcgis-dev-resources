## システム要件の確認

ArcGIS Pro は Microsoft Windows オペレーティング システムでサポートされています。

ArcGIS Pro をインストールするコンピューターがシステム要件を満たしているかどうかをご確認ください。

システム要件の詳細は[ヘルプ](https://pro.arcgis.com/ja/pro-app/latest/get-started/arcgis-pro-system-requirements.htm)をご参照ください。
