変換ツール (国内データ) for ArcGIS Pro の操作ヘルプです。

{{< list-docs section="docs" >}}

{{< callout type="info" >}}

変換ツール (国内データ) for ArcGIS Pro は、 [アドイン ツール](/pro/get-started/setup/user/addin-tool) からダウンロードすることができます。ArcGIS Pro のバージョンにあったファイルをダウンロードしてインストールし、ご利用ください。

{{< /callout >}}



### 既知の制限
- この「変換ツール (国内データ) for ArcGIS Pro」は ArcGIS Pro SDK を用いて開発したアドインとして提供しているため、各ツールを ModelBuilder や Python で実行することはできません。また、ArcMap と ArcGIS Pro では開発テクノロジーが異なるため、同一名のツールでも完全に同じ機能を提供するものではありません。
- ツールの実行時に表示されるプログレス バーに表示されるパーセンテージは、残りの処理時間を正確に表すものではありません。設定したパラメーターや入力データの違い等によって、変動する場合があります。
- 「変換ツール (国内データ) for ArcGIS Pro」は、対応バージョンが ArcGIS Pro 3.5 以前のものは JGD2024 には対応していません。そのため、JGD2024 のデータをインポートしたい場合は、ArcGIS Pro 3.6 および、対応バージョンが ArcGIS Pro 3.6 のアドイン ツールを使用してください。
