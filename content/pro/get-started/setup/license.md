ArcGIS Pro のライセンス形態には以下の 3 つがあります。
- [指定ユーザー](#指定ユーザー-ライセンス) (デフォルト)
- [単独使用 (SU)](#単独使用ライセンス-su)
- [同時使用 (CU) ](#同時使用ライセンス-cu)

{{< callout type="info" >}} 

ArcGIS Online の Creator、Professional、Professional Plus ユーザー タイプでは、指定ユーザー ライセンスとして提供されます。

{{< /callout >}} 

## 指定ユーザー ライセンス
ArcGIS Online 組織サイトのログイン アカウントを通じてライセンス認証されます。<br>
指定ユーザー ライセンスは、コンピューターではなくユーザーに割り当てられるため、ソフトウェアがインストールされており、動作環境を満たしているコンピューターであれば、どのコンピューターからでも自由に利用することができます (ArcGIS Pro 起動時に、ArcGIS Online のアカウントでサイン インを行います)。
<img src="https://s3-ap-northeast-1.amazonaws.com/doc.esrij.com/pro/get-started/setup/license-named.png" width="300" alt="指定ユーザー ライセンス">
指定ユーザー ライセンスで ArcGIS Pro をお使いいただくためには、2 つの条件を満たしている必要があります。
- ArcGIS Online の Creator、Professional、Professional Plus ユーザー タイプをご契約の方、または保守契約が有効な ArcGIS Desktop ライセンスを有すること
- ArcGIS Online の利用登録が完了していること

**ArcGIS Pro を初めて利用する場合は、まず ArcGIS Online の登録とライセンスの割り当てを行ってください。** 詳細は「[管理者の操作]({{% ref  "pro/get-started/setup/admin" %}} "管理者の操作")」をご参照ください。

既に利用登録済みの方や、ArcGIS Online 組織プラン、または ArcGIS アカデミックパックを契約されている方は、「[利用者の操作]({{% ref "pro/get-started/setup/user" %}} "利用者の操作")」の手順に従ってインストールを行ってください。

{{< callout type="info"  >}}

1 つの ESRI 顧客 ID につき、利用できる ArcGIS Online 組織サイトは 1 つです。

{{< /callout >}}

{{< callout type="info"  >}}

お持ちのライセンス数以上の割り当てはできません。

{{< /callout >}}

## 単独使用ライセンス (SU)
単独使用ライセンスでは、各コンピューターにソフトウェアをインストールし、ライセンス認証をしてご利用いただけます。

{{< callout type="warning" >}} 

単独使用ライセンスは、ArcGIS Desktop ライセンスとして購入し、ArcGIS Pro を利用する場合にのみご利用いただけます。<br>
なお、ArcGIS Desktop 単独使用 (SU) ライセンスのご契約でご利用いただける ArcGIS Pro 指定ユーザー ライセンスを単独使用 (SU) ライセンスに変換する機能は、2025 年 11 月 30 日をもって終了しました。

{{< /callout >}}

<img src="https://s3-ap-northeast-1.amazonaws.com/doc.esrij.com/pro/get-started/setup/license-su.png" width="450" alt="単独使用ライセンス">

## 同時使用ライセンス (CU)
同時使用ライセンスでは、ネットワーク上の 1 台のコンピューターに [ArcGIS License Manager]({{% ref "license-manager" %}} "ArcGIS License Manager") 製品をインストールして、そのネットワーク上で同時に使用できるソフトウェアのライセンスを一括管理します。<br>インストールするソフトウェア数に制限はなく、同時使用のライセンス数のみを管理します。

{{< callout type="warning" >}} 

同時使用ライセンスは、ArcGIS Desktop ライセンスとして購入し、ArcGIS Pro を利用する場合にのみご利用いただけます。<br>
なお、ArcGIS Desktop 同時使用 (CU) ライセンスのご契約でご利用いただける ArcGIS Pro 指定ユーザー ライセンスを同時使用 (CU) ライセンスに変換する機能は、2025 年 11 月 30 日をもって終了しました。また、2026 年 2 月末以前にリリースされたバージョンでは、既存の同時使用ライセンスを引き続きご利用いただけますが、2026 年 3 月 1 日以降にリリースされる ArcGIS Pro では、同時使用ライセンスはサポートされません。詳細は、[こちら](https://tech-support.esrij.com/arcgis/article/web/knowledge5996.html)をご確認ください。

{{< /callout >}}
