Survey123 で作成した調査票を保存し、公開、共有する方法について説明します。<br>

調査票の作成方法については「[調査票の作成](/survey123/get-started/prepare/create/)」をご参照ください。

### 調査票の保存とプレビュー
1. Web デザイナー画面右上の [保存 (ファイル マーク)] ボタンをクリックし、調査票を保存します。<br>
   調査票が保存されると、アイコンにチェック マークが追加されます。
 {{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/survey123/get-started/prepare/publish-save-survey-complete.png" caption="調査票の保存" width="600" >}}
1. 画面右上の [プレビュー (四角と三角のマーク)] ボタンをクリックし、調査票の実際の画面を確認します。<br>
   アイコンを選択することで、以下の画面解像度でプレビューすることができます。

    <ul>
        <li>デスクトップ</li>
        <li>スマートフォン</li>
        <li>タブレット</li>
    </ul>

    {{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/survey123/get-started/prepare/publish-preview-survey-screen.png" caption="調査票のプレビュー" width="800" >}}

### 調査票の公開
1. 画面右上の [公開] をクリックし、調査票を公開します。<br>
   調査票が公開されると、[公開済み] と表示されます。
   {{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/survey123/get-started/prepare/publish-publish-survey-complete.png" caption="調査票の公開" width="600" >}}

    {{< callout type="info" >}}

    調査票の公開前にフィーチャー レイヤーのスキーマを変更することができます。公開後に変更することはできません。<br>
詳細は<a href="https://doc.arcgis.com/ja/survey123/browser/create-surveys/publishsurvey.htm" target="_self">ヘルプ</a>をご参照ください。

    {{< /callout >}}

### 調査の共有
[調査の共有] タブでは、調査の共有範囲や調査ステータスなどを設定できます。<br>
1. [共同作業] をクリックし、共同作業ページに移動します。
1. [調査の共有] タブをクリックし、調査の公開範囲や調査のステータスを設定します。
    {{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/survey123/get-started/prepare/publish-survey-cowork-share.png" caption="[調査の共有]タブ" width="700" >}}

    {{< callout type="info" >}}

    [結果の共有] タブでは 、調査結果の共有範囲を設定できます。

    {{< /callout >}}

3. [URLのコピー] アイコンから調査票の URL を取得します。URL を調査者に共有することで、調査票にアクセスできるようになります。 <br>
    {{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/survey123/get-started/prepare/publish-survey-cowork-URL.png" caption="URL のコピー" width="650" >}}

    {{< callout type="info" >}}
   
    [QR を表示] アイコンをクリックし、[クリックしてダウンロード] をクリックすることで、調査の QR を入手できます。
  
    {{< /callout >}}
