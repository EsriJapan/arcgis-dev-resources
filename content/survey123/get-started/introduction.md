[Survey123](https://www.esrij.com/products/survey123/) の利用を開始する方のためのガイドです。

Survey123 は、現地調査の準備、実施、集計作業をシンプルかつ直感的な操作で行うことができる GIS アプリケーションです。Web ベースの作業により、現地調査の効率を向上させることができます。

### 利用の流れ

1. Survey123 Web デザイナーで、現地調査で使用する調査票を作成します。
1. ブラウザーや Survey123 フィールド アプリを利用し、現地調査を行います。
1. Survey123 Web サイトで、結果を表示します。

### 必要なライセンス

ArcGIS Online の[ユーザー タイプ](https://www.esrij.com/products/user-types/)によって Survey123 で利用可能な機能が異なります。

- **Creator**、**Professional**、**Professional Plus**: 調査票の作成、データの収集、結果の表示ができます。
- **Mobile Worker**: Web アプリおよびフィールド アプリでの回答の入力、結果の表示ができます。
- **Contributor**: Web アプリでのデータの収集、結果の表示ができます。
- **Viewer**: 結果の表示ができます。
- **ArcGIS Online のユーザー タイプを持っていない場合**: パブリックに公開されている調査のデータの収集、結果の表示ができます。

### フィールド アプリのインストール

{{< callout type="info" >}}

動作環境については「[システム要件](https://doc.arcgis.com/ja/survey123/get-started/systemrequirements.htm)」をご参照ください。

{{< /callout >}}

1. Android の場合は「Play ストア」、iOS の場合は「App Store」、Windows の場合は「Microsoft Store」アプリケーションを開きます。
1. キーワードに「Survey123」と入力して ArcGIS Survey123 を検索し、インストールします。
