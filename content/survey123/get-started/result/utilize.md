Survey123 で収集したデータを活用する方法について説明します。<br>

Survey123 では、収集されたデータをエクスポートして別のアプリケーションで使用できるようにしたり、Map Viewer で開いてシンボルを変更したり、他のレイヤーと重ね合わせて解析を実行したりできます。<br>
また、これらを新しいマップとして保存することで ArcGIS の Web アプリケーションで活用することもできます。<br>

### [データ] タブを開く
1. Survey123 Web サイトから、調査を開きます。<br>
   調査を開く方法については「[Survey123 Web サイトで調査結果を表示](/survey123/get-started/result/show/)」をご参照ください。
1. [データ] タブを開きます。
   {{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/survey123/get-started/result/utilize-survey-data-tab.png" caption="[データ] タブ" width="400" >}}

### データのエクスポート
Survey123 Web サイトでは、以下のファイル形式でデータをエクスポートできます。

<ul>
   <li>CSV</li>
   <li>Excel</li>
   <li>KML</li>
   <li>シェープファイル</li>
   <li>ファイル ジオデータベース</li>
</ul>

1. 画面中央の [エクスポート] から、出力形式を指定し、データをダウンロードします。
   {{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/survey123/get-started/result/utilize-survey-result-export.png" caption="[エクスポート]" width="400" >}}

### Map Viewer への追加


1. 画面中央の [マップ ビューアーで開く] をクリックします。
1. 画面右上の [新しいタブで開く] アイコンをクリックし、Map Viewer を起動します。 
 {{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/survey123/get-started/result/utilize-survey-result-mapviewer.png" caption="Map Viewer への追加" width="600" >}}
1. Map Viewer で操作を行います。<br>
   Map Viewer での操作は「[ArcGIS Online 逆引きガイド](/online/users-guide/)」をご参照ください。
