Survey123 Web サイトで調査結果を表示する方法について説明します。<br>

Survey123 Web サイト上では、収集された調査結果をグラフ形式やマップ上で確認することができます。

### Survey123 Web サイトで調査結果を表示
1. ArcGIS Online 組織サイトの右上に表示されている [アプリ ランチャー] → [Survey123] をクリックします。
2. 表示された調査一覧から、結果を表示する調査のサムネイルをクリックします。
   {{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/survey123/get-started/result/show-open-survey-result.png" caption="調査票の選択" width="200" >}}<br>

結果を表示できる各ページ の説明は以下の通りです。

#### 概要
コード総数や最終送信日などの調査結果の概要を確認できます。<br>
調査結果はリアルタイムで反映されます。
{{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/survey123/get-started/result/show-result-about-survey.png" caption="[概要]ページ" width="700" >}}

#### 解析
質問の集計結果を確認できます。<br>
各質問の右上から、集計結果の表示方法を変更することができます。主な表示方法は以下の通りです。
{{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/survey123/get-started/result/show-survey-analyse-graphs.png" caption="[解析]ページ" width="700" >}}
[カラム] (左上): 選択肢や数値などの質問の集計結果を縦棒グラフで表示します。<br>
[バー] (右上): 選択肢や数値などの質問の集計結果を横棒グラフで表示します。<br>
[パイ] (左下): 単一選択や数値などの質問の集計結果を円グラフで表示します。<br>
[マップ] (右下): 結果のジオメトリーをマップ上に表示します。<br>

#### データ
調査結果の一覧をマップとテーブルで確認できます。<br>
任意の結果を選択することで、各結果の詳細も確認、編集できます。
{{< figure src="https://s3.ap-northeast-1.amazonaws.com/doc.esrij.com/survey123/get-started/result/show-survey-data-table.png" caption="[データ]ページ" width="700" >}}
