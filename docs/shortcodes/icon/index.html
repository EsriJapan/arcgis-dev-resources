icon

Display an icon like

It uses :

  • glyphicons library (bootstrap).more info here
  • fontawesome library more info here

This icon shortcode will display an icon in your page.

Usage

Parameter Default Description
name required name of icon (see bellow)
size none size of icon, medium, xx-small, x-small, small, large, x-large, xx-large, 11px, 2em, 20%….

Tips : setting only the name as argument works too : {{<icon film>}} instead of {{<icon name="film">}}

Demo

{{< icon name="film" size="large" >}}
